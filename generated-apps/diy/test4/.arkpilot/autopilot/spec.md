# 手作·印记（Handcraft Imprint）产品规格书

## 产品概览

- **产品名称**：手作·印记
- **产品类别**：手作工坊会员管理与课程预约工具
- **核心用户场景**：陶艺/木作/皮具等手作工坊的会员到店签到、作品记录、课程预约，以及店长的沉睡会员提醒与库存扣减管理。
- **价值主张**：以温润陶土色与手写感为基调，把“会员档案、积分成长、课程预约、自动提醒、库存联动”串成一条手作工坊的日常运营线，让每位学员的作品与成长都有迹可循，让店长用更少精力记住更多人情。

## 目标用户

- **学员端**：手作爱好者，关注自己的作品记录、积分余额、课程预约。
- **店长端**：工坊主理人/店员，关注会员活跃度、沉睡提醒、课程排期、物料库存。

## 核心 Jobs-to-be-Done

1. **会员建档**：创建并查看会员档案，记录擅长工艺、个人简介、作品照片。
2. **积分成长**：通过签到、消费自动累计积分，查看积分明细与等级。
3. **课程预约**：浏览可约课程，选择日期时段预约，并在日历中管理行程。
4. **沉睡提醒**：系统自动识别超过设定天数未活跃的会员，生成提醒清单。
5. **库存扣减**：课程或消费发生时，按配方自动扣减对应物料库存。

## 导航模型

- **底部 Tab 导航**：首页、课程、会员、我的（共 4 个一级 Tab）。
- **二级入口**：
  - 首页 → 积分明细 / 沉睡提醒 / 库存管理 / 课程预约
  - 会员 Tab → 会员列表 → 会员档案
  - 课程 Tab → 课程日历 → 课程预约
  - 我的 Tab → 设置/关于
- **返回逻辑**：二级页面使用顶部返回按钮或系统返回手势回到上一级。

## 数据策略

- **本地优先**：使用 App 本地存储（关系型数据库 / 首选项）保存会员、课程、积分、库存、提醒数据。
- **真实空状态**：首次启动无预设演示数据，界面展示空状态引导用户创建会员或课程。
- **离线可用**：所有核心功能不依赖网络。
- **自动计算**：
  - 沉睡会员根据最后活跃日期与阈值（默认 30 天）自动计算。
  - 库存扣减在预约确认或消费确认时按配方联动扣减。
- **可选扩展**：未来可对接云端备份，但不在本次范围内。

## 视觉方向

- **关键词**：陶土色、手写感、温润、手作风、质朴、留白、自然肌理。
- **主色**：陶土暖棕 `#C17A53`、赤陶 `#A85D3C`、米白 `#F8F3EE`、深褐 `#3D2E26`。
- **辅色**：陶土阴影 `#E6D5C7`、手写墨 `#5C4A3D`、坯体白 `#FFFFFF`、提醒绯 `#C45A4B`。
- **质感**：轻微纸张纹理、不规则圆角、手写体标题、柔和投影。

## Target Device

- 屏幕分辨率：1272vp × 2756vp（宽 × 高）
- 设备类型：普通手机（非折叠机、非平板）
- 像素密度倍率：3x（即 HTML 设计稿 390px = 设备上 1170vp 物理像素，但 ArkTS 使用 vp 单位时 1px ≈ 1vp）
- 布局策略：内容区 max-width 约束在 390-420vp 内居中，禁止全宽铺满。**注意：Tab 导航栏等必须在底部，且若使用 `position: fixed` 必须限制宽度与 body 一致（如 `width: 100%; max-width: 390px; margin: 0 auto; left: 0; right: 0;`），严禁在 PC 浏览器上全宽拉伸。严禁生成在侧边（如右边/左边）**。
- 特殊注意：2756px 高度意味着屏幕很长，滚动区域充足，但需确保首屏关键内容（Header + 核心卡片）在顶部 600vp 内可见

## 功能验收标准

1. 能新建会员档案，填写昵称、手机号、擅长工艺、简介，并上传/选择作品图。
2. 能在首页完成一键签到，积分余额实时更新，并在积分明细中显示来源。
3. 能在课程日历查看本月课程，点击空白日期进入预约，点击已有预约可取消。
4. 能在预约页选择课程类型、日期、时段，提交后库存按配方扣减。
5. 能在沉睡提醒页看到超过阈值的会员，支持一键发送唤醒提醒（本地记录）。
6. 能在库存页查看各物料当前库存与最近扣减记录。
7. 所有页面支持空状态、加载状态、错误状态。

## 非目标

- 不实现真实支付、短信推送、云端同步。
- 不实现多门店/多账号角色权限。
- 不实现复杂财务对账与报表导出。

## 待确认问题

- 是否需要店长/学员双角色切换？（默认单账号本地管理，不区分角色）
- 签到积分规则是否可配置？（默认固定：签到 +5，消费 1 元 +1）
- 库存不足时是否允许预约提交？（默认不允许，需提示）
