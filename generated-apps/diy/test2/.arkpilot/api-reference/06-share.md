## Module Notes

- **No `@ohos.share` API exists.** The documented import module is `@kit.ShareKit`, which exports two namespaces: `systemShare` and `harmonyShare`.
- `UIExtensionAbility` is referenced in `systemShare` via `ShareAbilityResultCode` (values returned from UIExtensionAbility to control share panel behavior: ERROR/BACK/CLOSE), but the UIExtensionAbility definition itself lives in Ability Kit docs, not in Share Kit.
- `Share Kit` is the kit name (`@kit.ShareKit`), not an individual API symbol. Both `systemShare` and `harmonyShare` are imported from the same kit.
- `systemShare` is the primary API for typical share-to-other-app scenarios. Minimum API version: 4.1.0(11). Stage model only.
- `harmonyShare` covers device-to-device sharing (knock/gestures/sandbox receive). Minimum API version: 5.0.0(12). Stage model only.
- Key system capabilities: `SystemCapability.Collaboration.SystemShare` (for systemShare) and `SystemCapability.Collaboration.HarmonyShare` (for harmonyShare).
