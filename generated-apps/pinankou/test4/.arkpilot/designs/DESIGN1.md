# 平安扣 Pinankou 视觉设计规范（评审优化版 DESIGN1）

## 1. 设计语言与氛围

「平安扣」的视觉关键词是「清澈、安心、可控」。相比初版，本版强化了：

1. **色彩层次更丰富**：不再局限于单一 teal，而是引入「深海蓝 → 湖水绿 → 薄荷青」的渐变光谱，配合暖白与雾灰，避免 AI 模板常见的平涂感。
2. **材质更细腻**：大面积使用轻微渐变、毛玻璃、柔和阴影与 1px 微边框，让卡片有「悬浮在湖面」的通透感。
3. **组件更精致**：按钮使用双色渐变 + 内发光；列表项加入左侧 4px 状态彩条 + 微圆角；图表使用品牌渐变填充。
4. **情感化插画**：空状态与首屏 Hero 使用几何化的安全主题 SVG（盾牌、对勾、定位针），并严格附带 contextual title。

## 5. 页面特定视觉（优化）

### 5.1 首页

- Header：毛玻璃，左侧应用名 + 右侧通知铃铛。
- Hero 区：
  - 大标题「早安，安全员」20px weight 700；
  - 日期 Caption；
  - 背景使用微弱的品牌渐变装饰圆（仅视觉，无交互）。
- 三个 KPI 卡片横向滑动或等宽网格：巡检进度、待处理隐患、本周已整改。
- 快捷操作：三个 Pill 渐变按钮，带图标。
- 最近动态：列表项带状态彩条与徽章。

### 5.2 巡检页

- 顶部卡片：左侧进度环，右侧路线名称与完成数。
- 点位列表：
  - 已完成：左侧品牌色对勾圆形 + 标题划线浅灰；
  - 当前：左侧高亮数字圆形 + 渐变边框；
  - 未开始：左侧灰色数字圆形。
- 操作区：打卡按钮（主按钮）+ 「报隐患」文字链接。

### 5.3 隐患页

- 搜索框带搜索图标。
- 状态与风险筛选器为横向滚动的 Pill 标签，选中使用品牌渐变背景。
- 「+」悬浮按钮：品牌渐变圆形，带加号图标。
- 列表项左侧 4px 风险色条；右侧状态徽章。

### 5.4 隐患详情

- 顶部大图/照片区（若存在照片）或纯色品牌渐变区。
- 标题 + 状态/风险徽章同行。
- 信息网格：2 列，展示位置、负责人、截止日期、上报人。
- 描述卡片：白色圆角卡片。
- 时间线：垂直，节点为渐变圆环，已完成节点填充。
- 底部固定操作栏：三个按钮（更新进度、编辑、关闭）。

### 5.5 联系人页

- 搜索框 + 分组标签横向滚动。
- 联系人卡片：左侧圆形头像（名字首字 + 品牌渐变背景），中间信息，右侧圆形电话按钮。
- 紧急联系人置顶并带「紧急」徽章。

### 5.6 统计页

- 周期选择器为分段控制器（Segmented Control），选中项品牌渐变背景。
- 4 个 KPI 卡片 2×2 网格。
- 趋势图：折线图，使用品牌渐变填充区域。
- 风险分布：环形图或横向条形图，使用功能色。
- 底部「导出报告」主按钮。

## 6. 插画与 SVG

- 所有图标使用内联 SVG，首子元素为 contextual `<title>`。
- 空状态插画示例：
  - 无隐患：盾牌 + 对勾 + 微笑弧线；title="暂无隐患，当前状态安全"。
  - 无巡检点：定位针 + 虚线路径；title="今日暂无巡检任务"。
  - 无联系人：电话轮廓 + 加号；title="暂无联系人，请添加应急联系人"。

## 7. 响应式约束

- 内容最大宽度 420vp，居中，两边留白。
- 固定底部元素宽度与内容区一致，禁止全宽铺满。
- 触控目标 ≥ 44×44vp。
- 列表底部预留 90vp 安全区。

## 9. 与 DESIGN.md 的差异摘要

1. 品牌色扩展为深海蓝 → 薄荷青光谱，不再单一 teal。
2. 按钮、KPI 卡片、Tab 选中态均加入渐变与内发光。
3. 列表项圆角从 12px 增至 14px，增加左侧状态彩条。
4. Tab 选中图标从正圆改为圆角矩形，增加光晕。
5. 首页 Hero 加入装饰性渐变圆，强化品牌氛围。
6. 统计页图表明确使用品牌渐变填充。
